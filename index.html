Это моя не первая веб-страница. Я учусь в колледже. Эта страница создана в рамках прохождения заданий по информатике.

HTML позволяет создавать структуру веб-страницы.

С помощью разных тегов можно добавлять заголовки, параграфы, ссылки, изображения и многое другое.

CSS позволяет изменять внешний вид элементов.

Но на самом деле он умеет гораздо больше, если знать множество различных приёмов!

А благодаря JavaScript можно сделать сайт более динамичным и реализовать взаимодействие пользователя с ним.

Я уже знаю HTML и CSS достаточно хорошо. А JS только учу. Мне предстоит научится самому главному и интересному в создании сайтов!
